QUESTIONS

Frequently Asked

Are my PDF files safe? ▾
Yes, completely. All editing happens in your browser. Your PDF file is never uploaded to any server. When you close the tab, all data is immediately cleared.
Is there a file size limit? ▾
No strict limit. Very large PDFs (100+ pages) may take a few extra seconds to render, but there is no file size cap.
Can I edit existing text in a PDF? ▾
You can add new text on top of existing content. Direct editing of original PDF text requires the source file — this tool adds new annotations and text layers over the existing content.
Will it add a watermark? ▾
No watermark, ever. Your saved PDF is completely clean with only the annotations you added.
Does it work on mobile? ▾
Yes — the editor works on mobile browsers. For the best editing experience with fine annotations, a desktop or laptop is recommended.

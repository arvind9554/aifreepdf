Questions

Frequently Asked

Are my files safe? ▼
Yes — 100%. All processing happens locally in your browser using JavaScript. Your files are never uploaded to any server. We can't see them even if we wanted to.
Is there a file size limit? ▼
No hard limit from our side, but very large files (100MB+) may be slow depending on your device's RAM and CPU. Most PDFs work instantly.
Does it work offline? ▼
Once the page is loaded, all tools work offline since processing is done locally. You only need internet to load the page initially.
Will it add a watermark? ▼
Never. AIFreePDF adds no watermarks, no branding, no hidden metadata. What you compress is what you get.
